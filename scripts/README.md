# Scripts 目录

这个目录包含可复用的辅助脚本，用于数据提取、分析和可视化。

## 脚本说明

### get_all_content.py
批量提取项目中所有方法的完整上下文信息。

**用途**:
- 从解析后的项目中提取所有方法的上下文
- 生成包含方法签名、类信息、参数/返回值类型等的JSON文件

**使用方法**:
```bash
python scripts/get_all_content.py
```

**输出**: `tmp/commons-lang-all-content.json`

**适用场景**:
- 需要批量导出方法信息用于LLM测试生成
- 构建方法数据集
- 分析项目整体结构

---

## 注意事项

- 这些脚本可能包含硬编码路径，使用前请检查并修改
- 脚本依赖项目已完成解析（需要.pkl文件）
- 输出文件默认保存在`tmp/`目录
